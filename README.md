## Running the Program
1. Download zip file

2. Download the [Fairy-Stockfish engine](https://github.com/fairy-stockfish/Fairy-Stockfish/releases). Choose [fairy-stockfish_x86-64-bmi2.exe](https://github.com/fairy-stockfish/Fairy-Stockfish/releases/download/fairy_sf_14/fairy-stockfish_x86-64-bmi2.exe) from release 14. 

3. Install Python dependencies:
   
   Make sure you have Python 3 installed.
   Run the following command in the terminal:
   ```python
   pip install chess

4. Run main.py to:
   - Start the game.
